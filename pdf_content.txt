Tips and Tricks
§ Packet Filter and Application-level firewall can identify the device d 
based on IP address and (limited to the same subnet) on MAC address 
– Notice that when traffic is routed between different networks, MAC 
addresses are changed
§ Application-level firewall can identify the device d based on hostname.
§ Packet Filter and Application-level firewall can identify the network n 
based on network ranges
§ Application-level firewall can identify the network n based on 
top/second-level domains and subdomains
§ PEP is a DBMS Client, i.e. PEP uses DBMS API
§ PDP is a SIEM Client, i.e. PDP uses SIEM API
